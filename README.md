# Home-App - React Lab 🧪

Este repositório centraliza meus estudos e práticas dentro do ecossistema **React**. Ele serve como um agregador de exemplos aplicados, onde cada seção explora conceitos fundamentais e avançados da biblioteca.

O projeto é estruturado para facilitar a consulta de implementações de referência para diferentes problemas comuns no desenvolvimento front-end.

## 📁 Módulos de Estudo

Abaixo estão os principais tópicos abordados nesta aplicação, com links para suas respectivas documentações técnicas:

### 1. [Dark-Light Mode](./src/pages/dark-light/README.md)
Estudo focado em gerenciamento de estado global simples utilizando **Context API** e **useReducer** para alternância de temas.

### 2. [Chat Exercise](./src/pages/chat-exercise/README.md)
Prática de lógica de estado mais complexa, simulando um fluxo de chat. Explora a fundo a combinação de **Context** com **Reducers** para manter a integridade de estados interdependente.

### 3. [TanStack Query](./src/pages/tanstack/README.md)
Exploração do gerenciamento de estado de servidor. Aborda cache, sincronização de dados, invalidação de queries e consumo de APIs assíncronas de forma performática.

---

## 🚀 Tecnologias Utilizadas

- **React** (v18+)
- **TypeScript** para tipagem estática.
- **Vite** como build tool.
- **TanStack Query** para server state.
- **Vanilla CSS** e componentes UI customizados.
